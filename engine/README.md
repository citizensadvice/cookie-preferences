## Installation
Add this line to your application's Gemfile:

```ruby
gem "citizens_advice_cookie_preferences"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install citizens_advice_cookie_preferences
```
